Interface Description

The interface studied in this project is that of a Kitchenette with a coffee machine, this interface is publicly accessible and situated in an open area typically designated for graduate computer science students and professors. This study was focused on the coffee-making experience of users that may interact with this interface. The vital interactive components in this interface in relation to the studied coffee-making task are labelled cabinets and the actual coffee machine.
